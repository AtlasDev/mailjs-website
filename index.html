Redefine your mail.

MailJS replaces all programs you ever needed to set up an email server for your company. It's build for security, extensibility, usability and speed, and within five minutes you are able to receive mails from your precious customers.

So many features

MailJS is full of features, which make the four key points of MailJS possible, security, extensibility, usability and speed. All this together is what makes MailJS awesome.

Security
Security is a big thing in MailJS. The internet is a dangerous place.Because of this, we automatically protect all connections (HTTPS, ESMTP, etc..) with a free certificate from Let's Encrypt, like the one on this site! You can add an extra layer of security by activating Two Factor Authentication. We also store your mails encrypted on the server.

Extensibility
Earlier you had to use a SMTP client if you wanted to integrate your mailbox into your own program, which can be an awful lot of pain. This is not necessary in MailJS, because it has an full HTTP API out of the box. Want to stay updated on new emails? A socketIO server does that for you. Because we don't like giving away passwords, there is a complete Oauth 2.0 integration, because we don't like sharing passwords.

Usability
The usablity of MailJS is amazing, for the sysadmins and also for the end user. Sysadmins are not having an hard time setting up MailJS, because it is largely automated. The only thing he has to do is run two commands and MailJS takes care of everything else. Adding users, domains or mailboxes is done on the web, not in terminals. The hosted version is even easier! Never again looking trough outdated docs to fix an issue!

The user can, for example, have multiple mailboxes and domains under one account and even share them with others. No more relogging or sharing account information with other people!

Speed
MailJS is blazing fast. Running on Node.JS server-side, MongoDB as database and Redis as cache let's it outruns any other mailing system available on the market.
